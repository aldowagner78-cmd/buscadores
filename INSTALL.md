# Cómo instalar la app en tu celular

## Android (Chrome)

1. Abre la app en Chrome
2. Toca el menú de 3 puntos (⋮) arriba a la derecha
3. Selecciona "Agregar a pantalla de inicio" o "Instalar aplicación"
4. Confirma y listo - ahora tendrás un ícono en tu pantalla

## iPhone/iPad (Safari)

1. Abre la app en Safari
2. Toca el botón de compartir (□↑) 
3. Desplázate y selecciona "Agregar a la pantalla de inicio"
4. Toca "Agregar" - aparecerá el ícono en tu pantalla

**Nota:** La app debe estar subida a un servidor web (no funciona con file://)
Si usas GitHub Pages, Netlify, Vercel o cualquier hosting, funcionará perfecto.

Una vez instalada, se abrirá como app independiente sin la barra de Chrome.
